# Hryvnia Calculator

Welcome to the Hryvnia Calculator! This tool allows users to easily convert amounts to and from the Ukrainian currency, the Hryvnia (UAH).

## Features
- Convert UAH to other currencies.
- Convert other currencies to UAH.
- Display current exchange rates.

## Usage
1. Input the amount you want to convert.
2. Select the currency you want to convert to/from.
3. Click on the convert button to see the result.

## Installation
You can install the Hryvnia Calculator by cloning the repository:
```bash
git clone https://github.com/Red590777/calculator.git
```
